Welcome to RegionShop

RegionShop is a Bukkit Plugin. It aims to create a Shop which handles all the Economic influences a Shop can have (Taxes, Dynamic Market, Sales, Splittable Shops, Rents). On this Site you can find a Link to the Plugin inside Bukkit, a Link to the CI for Dev Builds and a Wiki Link so you can search for Guides and get into the usage of RegionShop.

The current State of the Plugin

* You can create Servershops which have unlimited Ressources
* The Servershops can have Signs which you can punch to sell/buy
* The Servershops also can have Showcases (floating Items) under each Sign
* The Items inside the Servershop can be configured as the example states below
* The Itemprices get recalculated based on their Config and Usage in the Shop
* You can make WorldGuard Regions to be your personal Shop
* You can create ChestShops with a Chest and a Sign

What will be in the reachable Future ?

* A refactor to get an external API so everybody can write Extensions to the Shop
* Rework Showcase Code since it seems to be buggy in some cases ( which i cant reproduce :( )
* Functional Rent modes (Rent split over Owners, Rent split via Itemusage, Rent split via Sell/Buy)

A little Lookout

* Splittable Shops (you share one Itemstorage and the sell/buys get splitted)
